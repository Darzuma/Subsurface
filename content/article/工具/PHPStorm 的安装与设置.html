PhpStorm 是一个集成开发工具，比 WebStorm 更易上手，功能更全，其插件的使用体验，比单纯的编辑器方便得太多！ PhpStorm 不用安装和调试大量插件，真正的、纯粹的开箱即用。JetBrains 的经典主题 Darcula(德古拉)，绝对是久经考验的最佳实践，人见人爱，花见花开啊！

PhpStorm 的安装

若电脑上安装有以往版本 PhpStorm ，会自动提示并自动卸载现有版本所有内容，无须担心直接安装会产生混乱。

网盘链接：https://pan.baidu.com/s/1nIuTDW0PjYqzwoIGWf_HeQ

安装后实际大小为 727M JBR x86 是JetBrain Runtime，32位系统需点选此项 选择 "I want to manully reboot later"

安装完成后，可直接使用（会有激活或试用提示），并不影响之后的破解。

PhpStorm 的激活

PhpStorm 的激活非常方便，直接输入激活码就行了，若之前安装过旧版 PhpStrom，重装后首次启动时会提示导入设置，选择 “Do not import settings”，然后点 “OK” 继续：

若之前有旧版 PhpStorm，会提示导入设置文件 启动后，选择 Activation code

激活码获取地址：PHPStorm 激活码

PhpStorm 的配置

默认情况下，PhpStorm 复杂而臃肿，启动非常缓慢，因此需要非常多的设置修改，关闭很多加载项。 次表面已将配置文件导出，下载后导入即可。

配置文件网盘地址：https://pan.baidu.com/s/1DskpZojsxapfCn0YuptwoA

启动后，在 File 选项卡下，选择 Import Settings 先点击 Select All，再点 OK

PhpStorm 的汉化

PhpStorm 的汉化和复原都非常方便！在安装目录内的 lib 目录内，删除英文包(resources_en.jar)，再把中文包(resources_cn.jar)拷贝进来即可。 目前汉化存在一个 bug，汉化后无法打开设置选项，复原英文可解决该问题。

中文包下载：resources_cn.jar

英文包下载：resources_en.jar

删除 resources_cn.jar，再把 resources_en.jar 拷贝回来即可复原
