请分析从输入地址到看到页面的过程

  1. 本地 host 文件
  2. DNS 查询
  3. 建立 TCP 连接(三次握手)
  4. 发送 http 请求（请求报文的格式？）
  5. 服务端：
    监听端口(浏览器默认监听80端口)
    拿到请求后处理路由和数据
  6. 服务端发送 http 相应
  7. 超时会关闭 TCP 连接（四次挥手）
  8. 浏览器解析响应数据
  9. 浏览器检查缓存，或者下载 css 和 js 媒体素材等，解析文件和素材
  10. 创建 DOM 树
  11. 渲染 DOM
  12. 执行 js
