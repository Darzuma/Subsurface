如何居中一个浮动元素

先清浮动，给父元素使用 .clearfix ，再使用弹性布局 

div.box{
  border:1px solid black;
  display: flex;
  justify-content: center;
  &.clearfix::after{
    display: block;
    content: '';
    clear: both;
  }
  div.floater{
    display: block;
    width: 100px;
    height: 100px;
    background-color: red;
    line-height:100px;
    text-align:center;
    float: left;
  }
}
