数组去重

方法一：用计数逻辑去重：创建一个空数组，将原数组元素加入新数组，加入时检查是否存在。 坑：数组遍历，避开 for ... in，尽量使用 for 或者 forEach 

Array.prototype.unique=function(){
    let tmp=[]
    this.forEach((item)=>{
        if(!tmp.join(',').includes(item))
            tmp.push(item)
    })
    return tmp
}
let arr=['a','b','a','c','c',1,1]
console.log(arr.unique());
